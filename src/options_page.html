Settings

BetterWeb is a chrome plugin that uses Machine Learning and Natural Language Process to help building a better, more rational web.

General

Extension status:

Controls whether the extension is enabled

Media Bias Panel

Status:

Controls whether news websites will have a panel showing their media bias

Information

This version of BetterWeb is built using data from Media Bias/Fact Check, a reputable website employing many journalists on internet media bias.

For more information, please visit our GitHub project page.

Machine Learning Panel

Status:

Conrtols whether machine learning feature is enabled.

Note: we will never collect your user data - and this plugin cannot connect to the internet. However, the information of the websites you are viewing (such as website URL, website content) would be processed in a machine learning algorithm locally on your computer.
